# macOS Portfolio

A portfolio website with a macOS-inspired interface, built with React and Vite.

## Overview

This project recreates the macOS desktop experience in a web browser, featuring:

- **Menu Bar** - macOS-style top toolbar
- **Dock** - Bottom application launcher with animated interactions
- **Finder** - Working file browser to explore projects and content
- **Window Management** - Draggable, resizable windows with macOS styling
